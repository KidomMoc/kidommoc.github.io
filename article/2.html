Essential HTML

km, Nov 12, 2017

HTML is a mark-up language. The content of web sites is stored in HTML. It consists of elements representing the content.

Element

An element may look like this:

<a href="http://kidommoc.com">km's home page</a>
|<--------start tag--------->|<--content-->|<--end tag-->|

As shown above, an element is made up by start tag, content and end tag. Tags are surrounded by angle brackets, and end tag has a slash following the former one. That between paired angle brackets is the name of the element.

Some elements have no content so the end tag can be omitted. If so, the start tag has a slash and a optional space before its latter angle bracket:

<br />

Elements can also have attributes. In the above example, element a has an attribute called href, and its value is "http://kidommoc.com". Note that the double quotations are a part of the value cause some values are absolutely number like 1. Universal attributes encompass id, class and style, used to indicating the id (must be unique) and the class, and the inline style sheet of this element.

Elements can be classified into block elements, inline elements and invisible elements. Block elements means a individual block and can exist alone, whereas inline elements must be in a block. Some elements must be unique in an HTML file.

Conventional Format

An HTML usually obeys the format as follows:

<!DOCTYPE html>
<html>

<head>
    <meta charset="utf-8" />
    <title> ... </title>
    ...
</head>

<body>
    ...
</body>

</html>

<html> is a unique block element as the root element, says, containing all other elements. Browsers will only handle those inside <html> element. Sometimes it takes lang attribute, which determines the language of the page.

<head> and <body> are both unique. Conventionally they contain those will only be aware of the browser and those will be shown to the users respectively. It's ok to put a <p> element in <header>, where it will work well, but doing so is not recommended.

<meta> takes the meta data of the page through attributes, usually only the charset with the value utf-8.

<title> determines what will be shown at the window and tab bar.

There is also an element named <link>, which is usually used to link something, most of time style sheet, through href attribute.

Block elements

  • <p>
    A paragraph. Any rebundant space, tab or line break will be omitted.
  • <hx>
    A title. `x` is a integer between `1` to `6` indicating the level.
  • <div>
    A universal container.
  • <header> (unique)
  • <footer> (unique)
  • <main> (unique)
  • <aside> (unique)
  • <article>
  • <section>
  • <ul>, <ol>
    Unordered and ordered list. The entries are <li>.
  • <hr>
    Makes a line.
  • <img>
    Inserts an image. Always takes a src attribute which indicating the URL of the image.

Inline elements

  • <a>
    Hyperlink. Use href attribute to determined where to link.
  • <strong>
    Presents importance.
  • <em>
    Presents the point.
  • <cite>
    Quote name.
  • <small>
    Side comment like author, copyright, etc.
  • <code>
  • <span>
    Universal text container.
  • <br>
    Line break. Always an empty element.

Script

There is another special element called <script>, which is used to insert Javascript or other scripts into pages.

It always takes a type attribute in MIME format indicating the type of script. For instance, <script type="text/javascript">.

Note that the end tag cannot be omitted.

Scripts can be embedded between <script> tags directly, or be referred from somewhere else through src attribute.

Comment

Comment in HTML is surrounded between <!-- and -->.

Escape

Like other programming language (though HTML is not actually a programming language), HTML has escape also to present some characters which cannot be input directly. For more infomation, please looks at here.

One more thing

HTML only takes content but not appearance. The latter is the responsibility of CSS. That's important.
